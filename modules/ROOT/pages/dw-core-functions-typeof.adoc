[[typeof1]]
== typeOf<T&#62;&#40;value: T&#41;: Type<T&#62;

Returns the basic type of a value.


A value's type is taken from its runtime representation, and will never be one of
the arithmetic types (intersection, union, `Any`, or `Nothing`) nor a type
alias. If the value has metadata present, this will be included on the result of
`typeOf`.

=== Parameters

[%header, cols="1,3"]
|===
| Name | Description
| `value` | A value.
|===

=== Example

This example identifies the type of several input values.

==== Source

[source,DataWeave,linenums]
----
%dw 2.0
output application/json
---
[ typeOf("A b"), typeOf([1,2]), typeOf(34), typeOf(true), typeOf({ a : 5 }) ]
----

==== Output

[source,JSON,linenums]
----
[ "String", "Array", "Number", "Boolean", "Object" ]
----

=== Example

This example shows that the type of a value is independent of the type it's
declared with.

==== Source

[source,DataWeave,linenums]
----
%dw 2.0
output application/json

var x: String | Number = "clearly a string"
var y: "because" = "because"
---
[typeOf(x), typeOf(y)]
----

==== Output

[source,JSON,linenums]
----
["String", "String"]
----
